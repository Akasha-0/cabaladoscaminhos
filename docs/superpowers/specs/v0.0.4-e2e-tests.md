## 1. Context

- ✅ Setup Playwright

---

## 2. Decisões de Design

| Decisão | Escolha | Rationale |
|---------|---------|-----------|
| Framework | **Playwright** | Melhor DX, screenshots, trace |
| Browser | **Chromium** | Padrão, mais confiável |
| Base URL | **http://localhost:3000** | Dev server |
| Parallel | **1 worker** | Simplicidade em CI |
| Screenshots | **On failure** | Debugging |

---

## 3. Test Scenarios

### 3.1 Auth Flow

| Scenario | Steps | Expected |
|----------|-------|----------|
| Login success | Open /login, fill form, submit | Redirect to / |
| Login invalid | Fill wrong credentials | Show error |
| Register | Open /registro, fill form, submit | Redirect to /login |
| Logout | Click logout button | Redirect to /login |

### 3.2 Dashboard Flow

| Scenario | Steps | Expected |
|----------|-------|----------|
| View dashboard | Login → visit / | Shows user name |
| Navigate to chat | Click chat link | Load chat page |
| View insights | Click insights | Show daily insight |

### 3.3 Protected Routes

| Scenario | Steps | Expected |
|----------|-------|----------|
| Access without auth | Visit /chat directly | Redirect to /login |
| API returns 401 | Fetch /api/credits | 401 Unauthorized |

## 5. Setup

### 5.1 Dependencies

```bash
npm install -D @playwright/test
npx playwright install chromium
```

### 5.2 Config

```typescript
// playwright.config.ts
import { defineConfig } from '@playwright/test';

export default defineConfig({
  testDir: './e2e',
  timeout: 30000,
  use: {
    baseURL: 'http://localhost:3000',
    screenshot: 'only-on-failure',
    video: 'retain-on-failure',
  },
});
```
