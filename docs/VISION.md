## 📜 Propósito do Projeto

### Missão

A **Cabala dos Caminhos** é um sistema de autoconhecimento espiritual que conecta múltiplas tradições místicas, oferecendo ao usuário uma compreensão profunda e integrada de sua jornada de vida.

### O Problema que Resolvemos

1. **Números sem contexto:** Usuários recebem números (caminho de vida = 11) sem compreender seu significado real no contexto de suas vidas.

2. **Tradições isoladas:** Sistemas espirituais como numerologia, astrologia e Odu de Ifá operam separadamente, sem reconhecer as correlações profundas entre eles.

3. **Falta de explicação educativa:** Conteúdo espiritual em português frequentemente falta clareza e contextualização, deixando usuários confusos.

4. **Widgets sem propósito:** Muitos aplicativos de espiritualidade apresentam ferramentas desconectadas que não contribuem para o autoconhecimento real.

### Nossa Solução

O Cabala dos Caminhos oferece:

- **Perfil Consolidado:** Uma visão completa que integra Numerologia Cabalística, Odu (Ifá), Astrologia, Numerologia Tântrica e Elementos
- **Explicações Contextualizadas:** Linguagem clara em português que conecta tradições e revela insights práticos
- **Preceitos e Orientações:** Recomendações acionáveis baseadas na análise integrada de múltiplos sistemas
- **Dashboard Minimalista:** Interface focada em clareza e autoconhecimento, não em abundância de ferramentas

## 🏛️ Arquitetura de Cálculos

### Sistema 1: Numerologia Cabalística

```
Cálculo: Soma dos dígitos da data de nascimento
Redução: Continue somando até obter um dígito (1-9)
Exceção: Números mestres (11, 22, 33) são preservados
```

**Princípio:** Os números mestres indicam espiritualidade elevada e não são reduzidos.

**Interpretação:**
- **1-9:** Características específicas da energia numerológica
- **11:** Intuição espiritual elevada, visão,_CHANNELING
- **22:** Mestre construtor, capacidade de manifestar realidades
- **33:** Mestre professor, compaixão infinita

---

### Sistema 2: Odu de Nascimento (Ifá)

```
Cálculo: Dia + Mês da data de nascimento
Se resultado > 16: some os dígitos novamente
Mapeamento: Resultado 1-16 para os 16 Odús do Ifá
```

### Sistema 3: Astrologia

```
Cálculo: Identificação do signo solar pela data de nascimento
Regência: Cada signo possui planeta regente e elemento
```

**Elementos e Regências:**

| Elemento | Signos | Energia |
|----------|--------|---------|
| 🔥 Fogo | Aries, Leão, Sagitário | Paixão, ação, transformação |
| 🌱 Terra | Touro, Virgem, Capricórnio | Estabilidade, materiallização |
| 💨 Ar | Gêmeos, Libra, Aquário | Comunicação, intelectual |
| 🌊 Água | Câncer, Escorpião, Peixes | Intuição, emoção, profundidade |

---

### Sistema 4: Numerologia Tântrica

```
Cálculo: Baseado na combinação de nome completo + data de nascimento
Foco: Identificação de chakras dominantes e excesso/deficiência energética
```

**Aplicação:**
- Identifica quais chakras estão mais ativos na energia vital da pessoa
- Aponta padrões de excesso (energia em excesso) ou deficiência (bloqueio)
- Orienta práticas específicas para equilibrar a energia dos chakras

## 🎨 Decisões de Design

### 1. Sem Algoritmos Falsos para Tarot e Mesa Real

**Decisão:** Tarot/Mesa Real NÃO devem ser algoritmos determinísticos.

**Justificativa:** 
- Tarot é uma ferramenta de autoconhecimento que depende da intuição do consulente
- "Mesa Real" sem contexto espiritual real é apenas entretenimento
- O sistema deve focar em correlações genuínas e insights práticos

**Implementação:** Remover funcionalidades que simulam espiritualidade sem propósito real.

---

### 2. Dashboard Simplificado (3-5 widgets principais)

**Decisão:** Interface minimalista com widgets focados em autoconhecimento.

**Widgets Prioritários:**
1. **Perfil Espiritual Consolidado** — visãocompleta de todas as correlações
2. **Insight Diário** — reflexão personalizada baseada no dia
3. **Preceitos e Orientações** — práticas acionáveis
4. **Correlações Ativas** — como os sistemas se conectam hoje

**Justificativa:** Usuário precisa de clareza, não de sobrecarga de informações.

---

### 3. Explicações Integradas no Conteúdo

**Decisão:** Todo número, símbolo e correlação vem com explicação contextual.

**Formato:**
```
[Número] + [Sistema] → [Significado no contexto do usuário]
```

**Exemplo:**
```
Caminho de Vida 11 + Escorpião + Alafia → 
"Sua intuição é sua ferramenta mais poderosa neste ciclo.
Pratique aterramento diário para equilibrar sua alta
sensibilidade espiritual."
```

---

### 4. Mobile-First para Acesso Cotidiano

**Decisão:** Interface otimizada para uso diário em dispositivos móveis.

**Prioridades:**
- Display de insights em tamanhos legíveis
- Navegação simplificada por seções
- Notificações para práticas e reflexões diárias

## 🎯 Conclusão

O Cabala dos Caminhos não é apenas mais um aplicativo de espiritualidade — é um sistema de **correlação integrado** que ajuda o usuário a compreender sua jornada espiritual através de múltiplas tradições.

A força do projeto está em mostrar as **conexões profundas** entre sistemas que normalmente operam isoladamente. Quando compreendemos que:

- O número 11 da numerologia ressoa com o Odu Alafia
- O signo de Escorpião (água) se conecta com o chakra Ajna (intuição)
- O planeta Vênus é regido por Oxum no Candomblé

...começamos a ver a **unidade subjacente** em todas as tradições espirituais.

**A missão continua:** Transformar dados espirituais em sabedoria aplicável.
