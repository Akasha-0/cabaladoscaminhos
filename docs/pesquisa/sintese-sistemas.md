# Sintese de Sistemas - Resumo Operacional

> Resumo canonico dentro de `docs/`.
> Use este arquivo como referencia rapida para sistemas sinteticos de referencia.

## Achados Que Permanecem Uteis

- Sistemas sinteticos bem-sucedidos criam um artefato visual central, nao apenas uma pilha de leituras.
- O usuario retorna quando o sistema responde ao "e agora?", nao apenas ao "quem sou eu?".
- Human Design mostra a forca de um framework de decisao; Gene Keys mostra a forca de uma arquitetura de transformacao.
- Linguagem proprietaria, coerencia visual e ciclo diario sustentam produto e comunidade.
- Para o Akasha, a sintese precisa parecer um sistema proprio, nao uma colagem de referencias.

- Use este resumo como base curta para principios transferiveis e implicacoes de produto.
- Quando precisar aprofundar a pesquisa, consulte os artefatos exploratorios ainda ativos em `.autonomous/`, sem recriar uma copia canonica fora de `docs/`.
