# Benchmark de Apps - Resumo Operacional

> Resumo canonico dentro de `docs/`.
> Use este arquivo como referencia rapida de benchmark competitivo.

## Achados Que Permanecem Uteis

- O mercado brasileiro responde bem a profundidade com linguagem nativa em pt-BR.
- O loop diario e mais valioso para retencao do que calculadoras de uso unico.
- O principal motivo de rejeicao em concorrentes maduros e paywall agressivo.
- UX limpa e foco de produto claro elevam satisfacao percebida mesmo com menos features.
- Nao ha concorrente direto para uma sintese real de cinco tradicoes.

## Como Usar Este Estudo

- Use este resumo como base curta para contexto de mercado e decisao de produto.
- Quando precisar aprofundar a pesquisa, consulte os artefatos exploratorios ainda ativos em `research/` ou `.autonomous/`, sem recriar este benchmark em duplicidade.
