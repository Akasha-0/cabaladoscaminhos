# Pesquisa - Ponte Documental

Esta pasta funciona como camada de resumo e navegacao da pesquisa consolidada em `docs/`.

## Regra de Uso

- Arquivos desta pasta devem concentrar os resumos duraveis usados para navegacao humana.
- Estudos exploratorios podem continuar em `research/` ou `.autonomous/`, sem duplicar docs canonicos.
- Quando um estudo virar referencia recorrente, sua versao curta deve existir aqui.
