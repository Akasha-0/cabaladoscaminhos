## Decisões-chave (6)

### D1 — Trio FASE 8 completo (HD, GK, Enneagrama)
**Decisão:** R-014, R-015, R-016 formam o trio de "sistemas de tipos modernos".
**Razão:** pesquisar em trio permite comparação direta (5 HD vs 3 GK vs 9
Enneagrama). Akasha herda insights de cada um sem se vincular a um único.

### D2 — Nomenclatura PT-BR para 9 Levels
**Decisão:** Akasha renomeia os 9 Levels (R-H) para nomenclatura PT-BR:
Presença, Clareza, Serviço, Tensão, Atribuição, Compensação, Ruptura,
Adicção, Dissolução.
**Razão primária:** usuário brasileiro.
**Razão secundária:** evitar trademarks R-H (Liberation, Social Value, etc.).
**Custo aceito:** perde reconhecimento internacional (Enneagrama é conhecido
mundialmente em círculos terapêuticos, mas Akasha é projeto brasileiro).

### D3 — NÃO usar 27 Subtypes (Naranjo)
**Decisão:** Akasha NÃO implementa 27 subtypes (9 tipos × 3 instintos).
**Razão:** Naranjo é qualitativo (auto-avaliação), não cabe na matemática
64 do Akasha. R-022 §1.1 proíbe claims sem base empírica.
**Alternativa:** usar "qual seu instinto dominante?" como pergunta
**contemplativa** dentro do Mandato (não como Pilar estrutural).

### D4 — Combinar 3 Tríades (Enneagrama) + 7 chakras (Tantra)
**Decisão:** Akasha usa 3 Tríades (Head/Heart/Gut) como **framework Pilar 3**
(Numerologia Tântrica que já opera sobre chakras hindus).
**Mapping:** Head ↔ Sahasrara (coroa), Heart ↔ Anahata (coração),
Gut ↔ Manipura (plexo solar). É isomorfia natural.
**Resultado:** Pilar 3 tem 7 chakras tradicionais + 3 Tríades como
framework pedagógico. Não é redundância — é **reforço** (3 Tríades
dizem COMO operar, 7 chakras dizem ONDE).

### D5 — Enneagrama é "tipo contemplativo" (vs HD/GK calculados)
**Decisão:** Akasha NÃO calcula o "tipo" do usuário a partir de mapa
astrológico (estilo HD). Akasha PEDE ao usuário para **narrar onde está
hoje** e mapeia nos 9 Levels. Isso preserva a filosofia contemplativa.
**Razão:** R-022 §3.1 proíbe "promessa de resultados". Enneagrama é
contemplativo — não promete "você é Type 5, então X".

### D6 — IP do Enneagrama é mais fraco que HD/GK
**Decisão:** Akasha usa MAIS Enneagrama do que HD/GK porque IP é mais
fraco (sem trademarks específicas de tipos).
**Trade-off aceito:** Enneagrama é **menos matemático** (sem 64 gates,
sem 88° arc). Akasha pode usar para framework pedagógico mas não para
Pilar algorítmico (que é o Pilar 2 Astrologia com HD/GK).

## Achados não-óbvios (5)

### F1 — Riso-Hudson Levels são a maior inovação
A estrutura **9 levels healthy→average→unhealthy** é o **único framework
no espaço espiritual moderno** que dá uma **camada temporal dinâmica**.
HD/GK têm "camadas" mas são **fixas** (definidas no nascimento).
Akasha DEVE usar este conceito — é o que falta em HD/GK e o que
diferencia Akasha como "sistema vivo, não estático".

### F2 — Origem é disputada (Ichazo claima revelação)
Ichazo claima revelação sufí em 1950s. Naranjo (psiquiatra chileno)
aprendeu com Ichazo. Críticos dizem que os 9 tipos podem ser derivados
de psicologia clínica (Karen Horney, Fromm) sem revelação. Implicação:
**fraco claim de IP** — revelação mística não é fonte verificável.

### F3 — Trifix reduz ruído (Naranjo)
Trifix = 3 fixações, 1 por center (Head/Heart/Gut). Exemplo: Type 5w4
com Trifix 5-4-7. Em vez de tentar ser "puro Type 5", aceita que a
pessoa tem 3 fixações (uma por center). É mais realista.

### F4 — 3 Tríades isomórficas a 3 chakras principais
Head/Heart/Gut do Enneagrama = Sahasrara/Anahata/Manipura do Hindu.
Akasha pode usar essa isomorfia para **reforçar** Pilar 3 (que já
opera sobre 7 chakras hindus).

### F5 — Enneagrama é o mais "IP-clean" dos 3
- HD: trademark registrada (Human Design, BodyGraph)
- GK: trademark registrada (Gene Keys, Hologenetic Profile)
- Enneagrama: sem trademark específica de "9 Tipos". Ichazo não registrou.
  R-H só tem RHETI (teste comercial).

Akasha pode usar **mais** do Enneagrama do que de HD/GK sem risco IP.
