## Contexto

Akasha tem **5 pilares** (Numerologia Cabalística, Astrologia,
Numerologia Tântrica, Odu de Nascimento, I Ching). Mas todos os
sistemas derivados de uma pessoa no tempo **NASCEM ESTÁTICOS**: o
bodygraph não muda, o tipo MBTI é fixo, a Mandala Akasha é imutável.

O que falta em **todos** os sistemas que estudei (Gene Keys, HD, MBTI)
é a **dimensão do tempo psicológico** — onde o usuário **ESTÁ AGORA**
em sua jornada interna. O Enneagrama é o único sistema moderno de tipos
que trata **tempo** como variável essencial através dos **9 Levels of
Development** + **direções Integration/Disintegration**.

validade fatorial**.

não ajusta). **Mesma vulnerabilidade que MBTI (Pittenger 2005).**

## Conclusão

**Enneagrama é o "termômetro de saúde" do Akasha.** Onde outros
sistemas são fotos (Gene Keys, HD, MBTI), Enneagrama é o **filme**.

Posição específica no stack Akasha:
- **Gene Keys (RQ-001)** = horizonte (Shadow→Gift→Siddhi, qual é o
  destino)
- **Human Design (RQ-002)** = gráfico (bodygraph, qual é a estrutura)
- **MBTI/Jung (RQ-003)** = estática psicológica (4 funções, qual é
  a função)
- **Enneagrama (RQ-004)** = dinâmica temporal (9 levels, qual é o
  estado AGORA)
- **Akasha Mandala** = síntese (5 anéis sobrepostos)

A **inovação do Akasha** é **combinar**:
1. Tipologia (estática) — Gene Keys / HD / MBTI
2. Dinâmica (temporal) — Enneagrama 9 levels
3. Multi-tradição (5 pilares) — síntese horizontal
4. Prática diária (ritual) — engajamento vertical
5. Mentor IA (acompanhamento) — feedback loop

Nenhum dos 4 sistemas RQ-001 a RQ-004 faz **todos os 5**. Akasha
sim, no papel. Implementação vem na Fase 5+.
