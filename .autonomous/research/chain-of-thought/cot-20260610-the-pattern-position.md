# COT: RQ-006 — The Pattern — posição Akasha

## Contexto
Pesquisa da Fase 0 sobre The Pattern (Lisa Donovan, 2017) para
definir a referência de **linguagem do Mentor Akasha** (Pilar 5).
Precedentes estudados: Gene Keys (RQ-001, síntese), Human Design
(RQ-002, UX/visualização), MBTI (RQ-003, base psicológica),
Enneagrama (RQ-004, dinâmica temporal), Co-Star (RQ-005, UX
restraint), Cabala (RQ-008, Pilar 1 esqueleto), Numerologia
Cabalística (RQ-009, Pilar 1 engine).

The Pattern é a **única das 7 estudadas que ataca a camada
linguística** — todos os outros se preocupam com cálculo,
visualização, síntese ou restraint, mas não com **como o texto
soa na cabeça do usuário**. Por isso é a referência para Pilar
5 (AI Mentor).

## Hipóteses
1. **H1:** The Pattern viraliza porque suprime jargão e usa
   linguagem íntima. **Confirmada** — Lisa Donovan escreveu
   pessoalmente 18 meses de conteúdo (Bustle 2022); reviews
   apontam especificidade emocional como chave ("scarily
   accurate" = Barnett Effect + prosa de alta qualidade).
2. **H2:** O modelo "7 Bonds" (soulmate, extraordinary, etc.)
   é bait emocional viral mas eticamente problemático.
   **Confirmada** — críticos chamam de "destino codificado"
   (Skeptic 2025).
3. **H3:** O AI Chat "In-Depth" (2024) é anti-padrão perigoso.
   **Confirmada** — APA Health Advisory 2025 + App Store
   reviews negativas imediatas sobre sycophancy bias.
4. **H4:** Crescimento orgânico sem marketing é replicável.
   **Parcialmente confirmada** — funcionou para Donovan
   (Maker Studios network + celebridades), mas o canal
   (tweets de famosos) não é replicável para Akasha. Akasha
   precisa investir em **conteúdo próprio** (blog, podcast).

## Conclusão
**Posição Akasha:** The Pattern é **referência de linguagem
do Mentor** (Pilar 5), não de UX nem de síntese. Donovan
inova em **como contar**, não em **o que contar**. Akasha
deve copiar:

1. **2ª pessoa, presente, confessional** — Mentor fala
   **com** o usuário.
2. **Suprimir jargão por padrão** — toggle "modo cabalista"
   para quem quer.
3. **Camada de tradução humana** (não 100% LLM) — curadoria
   de conteúdo ritualístico.
4. **3 pilares canônicos** — Essência / Temporalidade /
   Relacional.
5. **Bonds qualitativos (7 rótulos)** — sem determinismo
   ("alma gêmea"); linguagem mais cuidadosa ("afinidade
   eletiva", "afinidade em descanso").

Akasha **melhora** em 6 dimensões:
- **Transparência epistêmica** (cita fonte sempre).
- **Disclaimer de saúde mental** (visível, persistente).
- **Botão de crise** (CVV 188, SAMU 192, CAPS).
- **Limite de uso** (3/dia pago, 1/dia free).
- **LGPD by design** (Akasha é BR; The Pattern é CCPA).
- **Multi-tradição** (Akasha combina 5; The Pattern é 1).

Akasha **evita**:
- Esconder fonte ("scarily accurate" = Barnum trap).
- Bait emocional ("soulmate").
- AI people-pleaser (sycophancy).
- Dependência de endorso de celebridade.
